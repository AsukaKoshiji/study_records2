# テスト計画書


## 目的

学習記録アプリケーションにおいて、各機能が設計通りに操作することを担保することを目的とする。

## テスト対象

### バックエンド

- FastAPI
- SQLAlchemy
- MySQL
- REST API

### フロントエンド

- Next.js
- React
- Tailwind CSS


## テスト範囲

|単体テスト|API,サービス、画面コンポーネント単位の動作確認|
|結合テスト|Frontend ↔︎ Backend ↔︎ DBの連携確認|
|画面テスト|画面表示、入力、画面遷移|
|APIテスト|正常系・異常系レスポンス確認|


## テスト環境

|OS|macOS|
|Backend|FastAPI0.115.6|
|Frontend|Next.js 16|
|DB|MySQL8.4|
|コンテナ|Docker Compose|
|テストツール|pytest|

## 完了条件
以下を満たした場合、テスト完了とする。
　
- 全テストケース実施完了
- 致命的不具合なし
- 正常系テスト100%成功
- 主要異常系テスト成功
